# AI Research Assistant (Agent-Based System)

## 🚀 Overview
This project is an AI-powered research assistant built using FastAPI, LangChain, and OpenAI APIs. It uses agent-based workflows to perform multi-step reasoning, search, and summarization.

## 🧠 Features
- LLM-powered query processing
- Agent-based multi-step reasoning
- Tool integration for search and summarization
- FastAPI backend with async support

## 🛠 Tech Stack
- Python
- FastAPI
- LangChain
- OpenAI API
- FAISS / Chroma (if used)
